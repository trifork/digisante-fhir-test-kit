Welcome to the HL7 Switzerland FHIR Test Kit — a disposable HAPI FHIR server for experimenting with Swiss FHIR Implementation Guides.

This server is operated for testing purposes by/for HL7 Switzerland, the HL7 Affiliate Switzerland. It loads Implementation Guides on startup and exposes both this generic tester and profile-restricted CRUD pages.

It is built on the open-source HAPI FHIR stack (Apache 2.0). Data on this server is for testing only and may be reset at any time.
